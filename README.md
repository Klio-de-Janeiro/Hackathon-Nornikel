# 🪨 ImageLab — AI-анализ полированных шлифов

**Streamlit-интерфейс для параллельного анализа концентрации талька, поиска тёмных включений внутри сульфидов и классификации руды моделью ResNet18.**

---

## 📋 О проекте

**ImageLab** — это end-to-end система автоматической классификации руд по панорамным микрофотографиям полированных шлифов. Решение заменяет субъективную визуальную экспертизу геолога на объективный количественный анализ с помощью компьютерного зрения.

### 🎯 Решаемые проблемы
- **Субъективность оценки** — разные геологи по-разному интерпретируют срастания сульфидов
- **Трудоёмкость** — ручная сегментация занимает часы на один образец
- **Масштабируемость** — ручной анализ не справляется с партиями из сотен шлифов
- **Отсутствие количественной оценки** — визуальная оценка не даёт точных процентов

### ✨ Возможности
- 🔬 **Сегментация сульфидных включений** — разделение на обычные и тонкие срастания
- 💎 **Детекция талька** — количественная оценка доли тёмной рассеянной фазы
- 🧠 **Классификация руды** — автоматическое определение типа по экспертной логике:
  - Оталькованная руда (тальк > 10%)
  - Рядовая руда (преобладают обычные срастания)
  - Труднообогатимая руда (преобладают тонкие срастания)
- 📊 **Интерпретируемый результат** — цветовая маска + таблица метрик + текстовое заключение
- 📄 **Экспорт в PDF** — автоматическая генерация отчётов
- ⚡ **Параллельная обработка** — одновременный запуск нескольких алгоритмов

### 🔧 Компоненты

| Модуль | Назначение |
|--------|-----------|
| `app.py` | Главный интерфейс Streamlit, управление параллельными задачами, визуализация результатов, генерация отчётов |
| `image_processing.py` | Функции `process_talc_concentration` и `process_sulfide_inclusions` для параллельной обработки изображений |
| `ore_classifier.py` | Загрузка ResNet18, классификация руды (автоматический запуск при доле талька < 10% или принудительно через кнопку) |
| `sam_predictor.py` | Сегментация с помощью Segment Anything Model (SAM), запускается по кнопке пользователя для детального анализа |
| `models/` | Хранение предобученных весов: ResNet18 для классификации и SAM для сегментации |
| `Jupiter Notebooks/` | Экспериментальные ноутбуки для исследования и отладки моделей SAM и MatSAM |
| `Articles/` | Научные статьи и публикации, использованные при разработке и обучении моделей |
| `.streamlit/` | Конфигурационные файлы Streamlit для настройки интерфейса и развёртывания |

## 🚀 Запуск локально

### Требования
- Python 3.12–3.14 (рекомендуется 3.12)
- Git

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/Klio-de-Janeiro/Hackathon-Nornikel.git
cd Hackathon-Nornikel

# Создание виртуального окружения
py -3 -m venv .venv

# Активация окружения (Windows PowerShell)
.\.venv\Scripts\Activate.ps1

# Установка зависимостей
python -m pip install --upgrade pip
python -m pip install -r requirements.txt

# Запуск приложения
python -m streamlit run app.py
```

Откройте адрес, который Streamlit покажет в терминале (обычно http://localhost:8501).

## 🌐 Развёртывание в Streamlit Community Cloud

1. Форкните репозиторий на GitHub
2. Перейдите на [share.streamlit.io](https://share.streamlit.io)
3. Подключите ваш репозиторий
4. Укажите:
   - **Main file path:** `app.py`
   - **Python version:** 3.12 (в Advanced settings)
5. Нажмите **Deploy**

⚠️ **Важно:** Версию Python уже созданного приложения можно изменить только удалением и повторным развёртыванием.

## 📊 Результаты работы

### Входные данные
- Форматы: TIFF, PNG, JPEG
- Минимальное разрешение: 1024×1024 пикселей
- Поддержка панорамных снимков до 10 000×10 000 пикселей

### Выходные данные
- 🎨 **Цветовая маска** поверх исходного изображения:
  - 🟢 Зелёный — обычные срастания
  - 🔴 Красный — тонкие срастания
  - 🔵 Синий — тальк
- 📈 **Таблица метрик:**
  - Общая доля сульфидов (%)
  - Доля обычных срастаний (%)
  - Доля тонких срастаний (%)
  - Доля талька (%)
- 📝 **Текстовое заключение:**
  > "Руда классифицирована как оталькованная: содержание талька — 14%, преобладание тонких срастаний — 62%"
- 📄 **PDF-отчёт** с визуализацией и метриками

## 🎯 Технические характеристики

| Параметр | Значение |
|----------|----------|
| Время обработки | ≤ 5 минут на изображение 10 000×10 000 px |
